# 👥 Customer Segmentation using K-Means Clustering

## 📌 Project Overview

This project was completed as part of the **Thiranex Data Analytics Internship**.

The objective is to segment customers into different groups based on their **Annual Income** and **Spending Score** using the **K-Means Clustering** algorithm. Customer segmentation helps businesses understand customer behavior and create targeted marketing strategies.

---

## 🎯 Objective

- Perform customer segmentation using Machine Learning.
- Identify customer groups based on purchasing behavior.
- Analyze customer characteristics.
- Visualize customer segments for better business insights.

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset

**Dataset:** Mall Customers Dataset

### Features

- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

---

## 📊 Project Workflow

### 1. Import Libraries

Imported the required Python libraries for data analysis, visualization, and machine learning.

### 2. Load Dataset

Loaded the customer dataset using Pandas.

### 3. Data Exploration

- Checked dataset information
- Displayed first records
- Verified missing values
- Checked duplicate records

### 4. Data Visualization

Performed exploratory analysis using charts such as:

- Age Distribution
- Gender Distribution
- Annual Income Distribution
- Spending Score Distribution
- Correlation Heatmap

### 5. Customer Segmentation

Applied the **K-Means Clustering** algorithm using:

- Annual Income
- Spending Score

### 6. Cluster Visualization

Visualized customer clusters with cluster centroids.

---

## 📈 Results

The K-Means algorithm successfully divided customers into **five distinct segments** based on spending behavior and annual income.

### Customer Segments

- ⭐ High Income – High Spending (Premium Customers)
- 💰 High Income – Low Spending (Potential Customers)
- 🛍️ Low Income – High Spending (Value Customers)
- 💵 Low Income – Low Spending (Budget Customers)
- 👥 Medium Income – Medium Spending (Regular Customers)

---

## 💡 Business Insights

- Premium customers should receive exclusive offers and loyalty rewards.
- High-income customers with low spending can be targeted using personalized marketing campaigns.
- Budget customers can be engaged through discounts and seasonal offers.
- Customer segmentation enables businesses to improve customer satisfaction and increase sales.
