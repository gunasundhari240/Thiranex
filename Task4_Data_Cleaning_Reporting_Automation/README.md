# 🧹 Data Cleaning & Reporting Automation

## 📌 Project Overview

This project focuses on automating the data cleaning and reporting process using Python. The workflow includes handling missing values, removing duplicate records, correcting inconsistent data, generating summary reports, and creating visualizations for better business insights.

This project was completed as part of the **Thiranex Data Analytics Internship**.

---

# 🎯 Objective

The objective of this project is to automate the data cleaning and reporting workflow to improve data quality, reduce manual effort, and generate meaningful reports for analysis and decision-making.
