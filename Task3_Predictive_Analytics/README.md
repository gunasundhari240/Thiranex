# 📈 Predictive Analytics Using Historical Data

## 📌 Project Overview

This project focuses on predicting future retail sales using historical business data and machine learning techniques. The objective is to analyze past sales transactions, preprocess the dataset, build predictive models, compare their performance, and forecast future sales trends.

The project demonstrates an end-to-end predictive analytics workflow including data preprocessing, exploratory data analysis (EDA), feature engineering, regression modeling, model evaluation, and business insights.

---

## 🎯 Objectives

- Predict future retail sales using historical transaction data
- Perform data preprocessing and feature engineering
- Explore sales patterns through visualizations
- Build multiple regression models
- Compare model performance
- Forecast future sales using the best-performing model

---

## 📂 Dataset

**Dataset:** Global Superstore Dataset

The dataset contains historical retail transactions including:

- Category
- Region
- State
- City
- Market
- Sales
- Profit
- Discount
- Quantity
- Shipping Cost
- Order Priority
- Customer Segment
- Order Date

## 🤖 Machine Learning Models

Three regression algorithms were implemented and compared:

- Linear Regression
- Decision Tree Regression
- Random Forest Regression

---

## 📈 Model Evaluation

Models were evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

### Model Performance

| Model | R² Score |
|--------|----------|
| Linear Regression | 0.5248 |
| Decision Tree Regression | 0.6501 |
| Random Forest Regression | **0.7174** |

Random Forest achieved the highest prediction accuracy and was selected as the final forecasting model.
